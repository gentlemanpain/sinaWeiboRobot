# SinaWeiboRebot
新浪微博机器人，实现基本的转评赞功能.  

# 使用说明：  
--在Config.properties里, 先填写相关配置：  
----client_ID ：appkey  
----client_SERCRET ：app_secret  
----redirect_URI : 回调地址

# 缺点
--新浪未开放转发微博API，所以现在无法转发微博.
